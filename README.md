Repositorio centralizado con el material interactivo y guías de estudio para las ayudantías de Sistemas Operativos. El objetivo es facilitar la comprensión teórica y práctica de la gestión de recursos, planificación de procesos y concurrencia.

---

##  Acceso al Material Interactivo

Las guías de ayudantía están diseñadas como páginas web interactivas para que puedas estudiar directamente desde el navegador, revisar diagramas de Gantt y verificar soluciones paso a paso.

**[Ver Sitio Web de Ayudantías ](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/)**

## Cronograma de Ayudantías

| N° | Tema Principal | Contenido Clave | Material Web |
| :---: | :--- | :--- | :---: |
| **01** | **Planificación y Multiprogramación** | Conceptos de Tarea/Proceso, Mono-programación vs Multiprogramación, Batch, Time Sharing y Diagramas de Gantt. | [Ver Guía 1](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-01/) |
| **02** | **Scheduling y Tiempo** | Conceptos de Programa/Tarea/Proceso, Planificación de CPU, Interrupciones vs Excepciones, Temporizador (RTMAX), TICK y cálculo de Overhead. | [Ver Guía 2](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-02/) |
| **03** | **Gestión de Memoria Lineal** | MMU, Direcciones Lógicas/Físicas, Particiones Fijas/Dinámicas, Fragmentación (Interna/Externa), Algoritmos (First, Best, Worst Fit), Compactación y Swapping. | [Ver Guía 3](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-03/) |
| **04** | **Memoria Virtual y Reemplazo de Páginas** | Paginación, Tabla de Páginas, Page Fault, Algoritmos de Reemplazo (FIFO, LRU, NFU, LFU), Principio de Localidad, Working Set y Simulador Interactivo. | [Ver Guía 4](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-04/) |
| **05** | **Repaso Certamen 2** | Repaso completo de Memoria Lineal y Virtual, algoritmos de reemplazo, hiperpaginación, Working Set y ejercicios resueltos. | [Ver Guía 5](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-05/) |
| **06** | **Procesos UNIX/Linux** | Estructura del proceso (Zona U, Task, segmentos), modos de ejecución, segmentos compartidos, estados y ciclo de vida, `fork()`, Copy On Write, `exec()` y terminación. | [Ver Guía 6](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-06/) |
| **07** | **Interfaces y Procesos** | Kernel y niveles de gestión del SO, interfaces con el hardware (puertos vs interrupciones), Interrupciones/Excepciones (síncronas/asíncronas), Interrupt Masking, Contexto y Conmutación de procesos, modos System/User y TRAPS/System Calls (IA-32 vs IA-64). | [Ver Guía 7](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/ayudantia-07/) |
| **Bonus** | **Repaso Certamen 3** | Preguntas conceptuales, Verdadero/Falso y selección múltiple sobre Procesos UNIX/Linux e Interfaces y Procesos (kernel, hardware, TRAPS, system calls). | [Ver Bonus](https://GonzaMichea.github.io/Ayudantias-Sistemas-Operativos/bonus/) |
